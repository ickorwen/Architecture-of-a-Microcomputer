Types of Buses

Data Bus

Data refers to the information that the CPU processes and transfers to/from different components (like instructions, numbers, and variables).

Data can be in various forms (binary, text, etc.), depending on what the CPU is processing.

The CPU sends and receives data to/from RAM, storage, and peripherals. Data travels through the system via the bus.

Address Bus

An address is a specific location in memory where data is stored or retrieved.

Each location in memory has a unique address, which the CPU uses to access specific data.

The CPU uses the address bus to specify where in RAM or storage it wants to read from or write to.

Control Bus

The control signals manage the operations of the computer by directing how data flows and when specific actions are performed.

These signals are sent by the CPU to control the timing of data transfer and execution of instructions.

The CPU sends control signals to other components (like memory and I/O devices) to manage processes and ensure data moves through the system correctly.
