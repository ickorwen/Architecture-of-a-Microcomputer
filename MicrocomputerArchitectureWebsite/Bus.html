Introduction

The bus system connects all major components, allowing data to flow between the CPU, memory, and other hardware.

Types of Buses

  • Data Bus: Carries data between components.
  • Address Bus: Sends memory addresses from the CPU to components.
  • Control Bus: Sends control signals to coordinate operations.
