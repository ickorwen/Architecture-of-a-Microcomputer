Keyboard

A hardware device with keys that allow the user to type text, numbers, and commands.

Typically consists of alphanumeric keys, function keys, and control keys. It connects to the computer via USB, Bluetooth, or wireless methods.

The keyboard sends signals (key presses) to the CPU, which processes the input and may trigger a response, such as text appearing on the screen or a program being opened.

Mouse

A pointing device used to move a cursor on the screen and interact with graphical elements.

It typically has two buttons (left and right) and a scroll wheel. Modern mice can be wired or wireless.

The mouse sends movement and button-click data to the CPU, which processes the input to perform actions like selecting, dragging, or clicking on items on the screen.

Monitor

A screen used to display visual output from the computer, such as text, images, and videos.

The CPU sends processed visual data to the monitor, which displays the output for the user. This could be anything from system information to multimedia content.

Speaker

A device that converts digital audio data into sound that can be heard by the user. Speakers can be built into the computer or external (wired or wireless). They can vary in quality, from basic sound output to high-fidelity audio systems.

The CPU processes audio data and sends it to the speakers, which then produce sound output, such as system alerts, music, or game audio.
